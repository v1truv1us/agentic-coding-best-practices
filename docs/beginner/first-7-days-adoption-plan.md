# First 7 Days Adoption Plan (Software Architect)

## Day 1 — Baseline Controls
- Configure permissions + sandbox baseline
- Define blocked paths/secrets policy
- Confirm approval mode for risky commands

## Day 2 — Plan/Spec Discipline
- Use spec template on one real task
- Require plan review before implementation

## Day 3 — Verification Gates
- Ensure lint/type/tests/security checks run in workflow
- Add explicit acceptance criteria to task prompts

## Day 4 — Context & Memory
- Refine CLAUDE.md
- Define context handoff format for long tasks

## Day 5 — Subagent Pilot
- Split one task across coordinator + one worker subagent
- Validate merge/integration protocol

## Day 6 — Reusable Assets
- Convert one repeated activity into skill/command
- Add to templates/playbooks if reusable

## Day 7 — Review & Calibrate
- Run post-week retro:
  - what failed repeatedly?
  - what should be documented?
  - what permissions should tighten?

Deliverable: short adoption report + next 30-day plan.
